Soso Valentines
================
Aiko and Laura's c++ app!

Building and running
--------------------
```bash
git clone --recursive git@github.com:laurasher/SosoValentines.git
```

```bash
cd Cinder/xcode
./fullbuild.sh

cd ../lib/Cinder/xcode
./fullbuild.sh
```
